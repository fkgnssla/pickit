## 프로젝트 소개
```
Pickit 은 블록체인 투명성 기반 온라인 투표 앱 서비스입니다. 🗳️

다양한 서바이벌 프로그램 투표 내역을 블록체인 기술을 이용하여 기록합니다. 
모든 투표 내역은 수정 및 삭제가 불가능하며 공정하고 투명한 투표가 가능합니다. 
```
